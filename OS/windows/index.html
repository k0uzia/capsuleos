  • Windows 98 — shell classic
  • Windows ME — shell classic
  • Windows 2000 — shell NT
  • Windows XP — Luna
  • Windows Vista — Aero Glass
  • Windows 7 — Aero 7
  • Windows 8 — Metro
  • Windows 8.1 — Metro + Démarrer
  • Windows 10 — Fluent moderne
  • Windows 11 — référence complète
