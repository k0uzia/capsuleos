# Notes CapsuleOS

Ce dépôt simule plusieurs systèmes d’exploitation dans le navigateur.

## Arborescence Linux FS

- `home/` — skins par distribution
- `usr/lib/capsuleos/` — noyaux JavaScript
- `usr/share/capsuleos/` — ressources partagées
- `home/public/` — **ce dossier** (home utilisateur unique)
