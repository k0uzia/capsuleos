# OS iOS (CapsuleOS)

## Arborescence

| Zone | Chemin |
|------|--------|
| Façade | `OS/ios/` |
| Branding portail | `assets/brands/ios.webp` (carte accueil ; pas de dossier icons dédié complet — voir `usr/share/capsuleos/branding/icons/README.md`) |

## Extension

Suivre `os-stub` pour nouvelles versions iOS : structure sous `OS/ios/<version>/`, factoriser JS vers `usr/lib/capsuleos/shells/ios/` quand le volume le justifie.

## Distinction

- **macOS** → `os-macos` (Finder, Sonoma)
- **iOS** → cette fiche (mobile, springboard, etc.)

## Rôles fréquents

`role-designer`, `role-web-designer`, `role-graphic-artist`.
