# Thème global CapsuleOS

| Fichier | Contenu |
|---------|---------|
| `reset.css` | Normalize + règles de base site |
| `variables.css` | Agrégateur (`variables-portal.css` + `../linux/variables-apps.css`) |
| `variables-portal.css` | Tokens portail (homepage), layout, couleurs Capsule |
| `../linux/variables-apps.css` | Tokens apps Linux (Mint, Nemo, Firefox, menus, thème clair) |

### Facades racine

`style/reset.css` et `style/variables.css` réimportent ce dossier pour compatibilité des skins non migrés.
